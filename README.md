# Vue 3 + Vite

- Módulo de Ponto

Registrar entrada e saída de turno.

Mostrar histórico de batidas com data/hora.

Pode armazenar tudo em LocalStorage (pra não depender de backend real no GitHub Pages).

Módulo de Lembretes / Passagem de Turno

Operador anota informações do turno (avisos, incidentes, observações).

O próximo turno consegue visualizar o que foi anotado.

- Módulo de Escala

Exibir a escala semanal/mensal (com os turnos: manhã, tarde, noite, madrugada).

Permitir solicitar troca de turno, que fica como “pendente” até o outro operador aceitar.

Sidebar (menu lateral)

Itens:

Registro de Ponto

Passagem de Turno

Escala

Banco de Horas (simulado)
